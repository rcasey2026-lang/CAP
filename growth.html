My Growth

This page is about my growth...


I believe this work represents how I have become a student who pays greater attention to detail. I believe that this project shows the beginning of my computer science journey, and I have grown to create projects that I am more proud of and that represent me as a student more. The bridge between coding and how the user interacts with the product has been hardest for me to master. For example, I will spend all of my time trying to code a project, but find it hard to think about how the user will interact with the project and adapt my code to fit it. I feel that I did not put my greatest effort into this project, but I have grown to put more effort and thought into my projects after reflecting on this one. After putting a greater effort into my work, it has directly reflected how successful I am in this class. Knowing what I know now, I would change a lot about this project and make it more useful to the user.
